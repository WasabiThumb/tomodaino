Personality type explorer for Tomodachi Life: Living the Dream™

What is this?

In Tomodachi Life: Living the Dream , you can create Miis with 5 varying personality traits. 4 of these traits decide, among other things, the "personality type" of the Mii. This system is unintuitive and it takes a long time to try several combinations in-game, so this site exists to play with the personality system in real time with no restrictions!


How do I use this?

This is a desktop-style web application with draggable widgets and a pannable/zoomable workspace. Interact with the input widget to set the personality traits and see the resulting personality type displayed in the output widget. In the future, there will be an option to add/create extra widgets such as graphs.


How does this work?

Honestly, I don't know! The algorithm was datamined from the game over the course of an hour in the form of a 2kB lookup table. This lets the website match the game exactly, but doesn't provide much insight into the underlying relationship. Here's the entire LUT as an image!
